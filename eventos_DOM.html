Acessando e manupulando o DOM - Métodos Eventos e Funções

MÉTODOS

Fazem a conexão entre o JS o HTML, o método precede a função e os eventos

Os principais Métodos JS que utilizamos no DOM são:

  1. addEventListener() - Espere algum evento para ser executado, monitora um evento

    Sintaxe: variavel.addEventListener('método/evento', nomeFunção);

  2. createElement() - Cria uma tag/elemento no doc HTML

    Sintaxe: var variável = document.createElement('Nometag'); Exemplo

  3. setAttribute() - Insere um atributo a variável criada

    Sintaxe: variavel.setAttribute('atributo', 'nomeAtributo');

  4. appendChild() - Adiciona um elemento criado ao final do documento

    Sintaxe: variavel.appendChild(elemento);

FUNÇÕES

Função é um conjunto de códigos que vão ser executadas SÓ QUANDO O EVENTO OCORRER, elas são executadas assim que as CHAMADAS ou com a decorrência de um EVENTO. Uma função pode receber PARÂMETROS e retornar um RESULTADO

  • Sintaxe 1

    function nomeAção() {

    BLOCO

    }

  • Sintaxe 2

    function nomeAção(parâmetros) {

    BLOCO

    }

  • nomeFunção(arg1, arg2) {

    return arg1 + arg2

    }

EVENTOS

Os principais Eventos JS que utilizamos no DOM são:

  1. mouseenter() - Evento acionado quando o mouse entra em um campo
  2. mouseout() - Evento acionado quando o mouse sai de um campo
  3. mousemove() - Evento acionado quando o mouse se movimenta em um campo, por exemplo uma div
  4. mousedown() - Evento acionado quando se aperta o mouse dentro de um campo
  5. mouseup() - Evento acionado quando se solta o mouse dentro de um campo
  6. click() - Evento acionado quando se clica o mouse dentro de um campo
  7. dlbclick() - Evento acionado quando se clica 2 vezes o mouse dentro de um campo
  8. mouseleave() - Evento acionado está TOTALMENTE FORA de um campo
  9. mouseover() - Evento acionado quando um dispositivo ponteiro é movido para o elemento que esteja escutando ou para um de seus filhos.
  10. select() - Evento acionado quando se seleciona um elemento
