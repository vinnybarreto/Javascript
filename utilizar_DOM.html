Acessando e manupulando o DOM

Seletores mais comuns

  1. document.getElementsByTagName()[] - Seleciona um determinado elemento através do seu nome

    Exemplo: document.getElementsByTagName('p')[0] - Seleciona o primeiro parágrafo do documento
    document.getElementsByTagName('p')[1] - Seleciona o segundo parágrafo do documento

  2. document.getElementById('meu-item-id') - Seleciona um determinado elemento através do id
  3. document.getElementsByName()[] - Seleciona um determinado elemento através do nome
  4. document.getElementsByClassName('.todos-itens-class')[] - Seleciona um determinado elemento através do nome da classe

Seletor mais usado

document.querySelector

  • document.querySelector("tag") - Seleciona a tag passada como parâmetro
  • document.querySelector("#id") - Seleciona o id passado como parâmetro
  • document.querySelector(".class") - Seleciona a classe passado como parâmetro

document.querySelectorAll

  • document.querySelectorAll("tag") - Seleciona todas tags as passadas como parâmetro
  • document.querySelectorAll("#id") - Seleciona todas os ids passadas como parâmetro
  • document.querySelectorAll(".class") - Seleciona todas as classes passadas como parâmetro
